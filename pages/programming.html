Chat App

Socket.IO, Express, JavaScript

This chat app uses Socket.IO - a library that enables event based communication between a client and server. I use Node.js to create an HTTP server and the Express app listens for this on http://localhost:3000. Once the server is running, the HTML (which stores the client side scripts) can be accessed on this port (3000). Any browser that goes to this port can connect to the chat server and send messages back and forth. The css is influenced by 'night-mode' colors to encourage comfortable use in low light. Check it out on GitHub.

Nobel Laureate App

React Native, Nobel Laureate API

This application uses React Native to build a cross-platform app for exploring the Nobel Laureate API. Users can navigate two stacks: 1 for recipients (organized alphabetically) and 1 for awards (organized by year). Each screen uses the useEffect hook to fetch data from the API and populate the cards with JSX. Check it out on GitHub.

Farmer Simulator Game

Unity, C#

This Farmer Simulation Game is an exercize in C# and programming a player controller to move through the game area and fire projectiles based on keyboard input. It also creates an array of farm animals that are sorted at random and sent running across the screen. The objective is to feed the animals by throwing them a banana before they run off-screen. Check out the code on GitHub.
